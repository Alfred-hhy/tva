## 💡 学习建议

### 对于 C++ 初学者
1. 先阅读详细文档的第二部分，了解 C++ 项目基础
2. 不要被模板语法吓倒，先理解概念
3. 多运行示例代码，观察输出
4. 从简单的测试程序开始修改

### 对于密码学初学者
1. 先理解秘密共享的基本概念（图解文档第1节）
2. 不需要深入数学细节，理解直觉即可
3. 通过运行代码观察秘密共享的效果
4. 逐步理解为什么需要不同类型的共享

### 对于 MPC 研究者
1. 先阅读原始论文了解理论基础
2. 对照论文和代码理解实现细节
3. 关注协议层的实现（详细文档 3.2.2节）
4. 研究性能优化技术（详细文档 6.3节）
